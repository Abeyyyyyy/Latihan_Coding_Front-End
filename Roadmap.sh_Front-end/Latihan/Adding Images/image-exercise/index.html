Galeri Gambar

Berikut adalah galeri gambar sederhana dengan berbagai eksperimen atribut gambar.

Penjelasan Eksperimen

  • Gambar Normal: Menggunakan width dan height yang sesuai.
  • Hanya Width: Height akan disesuaikan secara otomatis untuk mempertahankan rasio aspek.
  • Width dan Height Tidak Cocok: Gambar akan diregangkan atau dipotong untuk memenuhi dimensi yang ditentukan, yang dapat membuat gambar terlihat tidak proporsional.
  • Gambar Rusak: Ketika src tidak valid, browser akan menampilkan teks alt sebagai pengganti gambar.
  • Gambar Dekoratif: Menggunakan alt="" karena gambar ini hanya untuk dekorasi dan tidak memberikan informasi penting. Ini membantu screen reader mengabaikan gambar yang tidak esensial.
