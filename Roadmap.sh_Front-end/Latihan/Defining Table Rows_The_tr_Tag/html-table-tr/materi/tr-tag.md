# Mendefinisikan Baris Tabel: Tag `<tr>`

Tag `<tr>` (table row) digunakan untuk mendefinisikan **baris** dalam tabel HTML.  
Setiap `<tr>` berisi sel data (`<td>`) atau sel header (`<th>`).

---

## Struktur Dasar

```html
<table>
  <tr>
    <td>Data 1</td>
    <td>Data 2</td>
  </tr>
  <tr>
    <td>Data 3</td>
    <td>Data 4</td>
  </tr>
</table>
