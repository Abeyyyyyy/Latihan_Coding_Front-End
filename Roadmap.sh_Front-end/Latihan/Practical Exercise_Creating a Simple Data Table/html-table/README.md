# Latihan HTML Table

Repositori ini berisi latihan HTML tabel dasar hingga menengah, meliputi:

- Penggunaan `<table>`, `<tr>`, `<th>`, dan `<td>`
- Penerapan atribut `colspan` dan `rowspan`
- Studi kasus tabel nilai siswa dan perbandingan produk

Cocok untuk pemula yang sedang mempelajari HTML dasar.
